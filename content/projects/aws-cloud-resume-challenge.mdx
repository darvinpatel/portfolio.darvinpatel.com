## Overview

- Developed and deployed a personal resume website using AWS services such as S3, CloudFront, and Route 53 for hosting and domain management
- Implemented a serverless backend using AWS Lambda, API Gateway, and DynamoDB to create a dynamic visitor counter
- Automated infrastructure deployment with AWS CloudFormation (IaC), enabling seamless resource management and updates
- Established CI/CD pipelines using GitHub Actions to automatically test, build, and deploy code changes to the AWS environment
- Integrated CloudFront for content delivery, optimizing website performance with caching and SSL termination
- Demonstrated cloud proficiency by using IAM roles and policies to ensure secure access and operations across AWS services
